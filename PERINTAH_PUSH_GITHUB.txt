# PERINTAH PUSH KE GITHUB

## LANGKAH 1: CEK STATUS
git status

## LANGKAH 2: ADD SEMUA FILE
git add .

## LANGKAH 3: COMMIT
git commit -m "Add deployment documentation and Railway fixes"

## LANGKAH 4: PUSH KE GITHUB
git push origin main

---

## JIKA PERLU LOGIN:
- Username: GitHub username Anda
- Password: Gunakan Personal Access Token (bukan password GitHub)

---

## COPY-PASTE SEMUA SEKALIGUS:
git add . && git commit -m "Add deployment documentation and Railway fixes" && git push origin main
