## ğŸ”§ TROUBLESHOOTING

### **Problem: Bot tidak merespon**
**Solusi:**
- Pastikan bot sudah running di backend
- Cek apakah bot token sudah benar
- Restart backend: `python app.py`

### **Problem: Payment tidak ter-verify**
**Solusi:**
- Pastikan admin sudah klik Verify button
- Cek dengan `/pending-payments`
- Hubungi admin jika sudah lebih dari 24 jam

### **Problem: Foto tidak terkirim**
**Solusi:**
- Pastikan kirim sebagai **Photo**, bukan File
- Cek ukuran foto (jangan terlalu besar)
- Coba kirim ulang

### **Problem: Command tidak dikenal**
**Solusi:**
- Pastikan mengetik command dengan benar (case sensitive)
- Gunakan `/help` untuk lihat semua commands
- Pastikan bot sudah update dengan versi terbaru
